Get Started

The beta version only consists of CSS files, SVG icons and fonts.

Downloads

Unminified: 0.2.7

Introduction

Two CSS files, rocket.[brand].css and rocket.[brand].fonts.css, are required to be included in head section of your HTML or templates. For example, if your project uses Coles themes/styles, then the names are rocket.coles.css and rocket.coles.fonts.css.

rocket.[brand].css defines all styles of UI components whereas rocket.[brand].fonts.css contains references to the fonts. Without rocket.[brand].fonts.css, components can still work. However, the fonts are only rendered according to user's devices.

By default, font files are in assets/fonts directory. In the case that you'd like to deploy them separately, edit paths in rocket.[brand].fonts.css.

Example of head section
<head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">
    <link rel="stylesheet" type="text/css" href="rocket.coles.fonts.css">
    <link rel="stylesheet" type="text/css" href="rocket.coles.css">
</head>
